# 🌦️ WeatherSphere

### Real-Time Weather Intelligence at Your Fingertips

WeatherSphere is a modern weather application that provides real-time weather information, 5-day forecasts, location-based weather updates, dynamic weather themes, and a clean glassmorphism user interface.

## ✨ Features

* 🌍 Search weather for cities worldwide
* 📍 Use current location with Geolocation API
* 🌡️ Real-time temperature updates
* 🤗 Feels Like temperature
* 💧 Humidity information
* 🌬️ Wind speed tracking
* 📈 Atmospheric pressure
* 👁️ Visibility monitoring
* 🌅 Sunrise & Sunset timings
* 📅 5-Day Weather Forecast
* 🎨 Dynamic weather-based themes
* 🕒 Recent search history
* 📱 Fully responsive design
* ⚡ Fast and lightweight

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/KaustubhDeshmane/WeatherSphere.git
```

### Navigate to Project

```bash
cd WeatherSphere
```

### Add API Key

Open `script.js`

Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with:

```javascript
const API_KEY = "YOUR_OPENWEATHER_API_KEY";
```

### Run Application

Open:

```text
index.html
```

in your browser.
